### Temp Ui - node server

###### Supported OS
- Linux
- Windows

###### Required packages
- Node.js          v12.16.2
- Npm              6.14.4
- Mongodb          v4.2.6

###### Used node packages
- Express          4.17.1 
- Express-session  1.17.1
- Bcryptjs         2.4.3 
- Body-parser      1.19.0 
- Connect-flash    0.1.1
- Cookie-parser    1.4.5 
- Mongoose         5.9.10 
- Passport         0.4.1
- Passport-local   1.0.0 
- Pug              2.0.4

###### Run setup:
`npm install`
###### Run mongodb before run the app:
`sudo mongod` || `sudo service mongod start` || `systemctl start mongod`
###### Run the app:
`npm run dev`
###### Browse:
App will run at: `localhost:3000/`
###### Default user:
`username: admin`  
`password: 123456`
